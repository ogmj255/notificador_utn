🎓 Notificador de Horario UTN

Sistema automatizado para notificaciones diarias de clases por WhatsApp

✅ Sistema activo y funcionando

📡 Endpoints Disponibles:

GET /enviar-notificacion
📅 Envía mensaje matutino con horario completo del día
GET /recordatorio
🔔 Envía recordatorio con opciones de respuesta interactivas
POST /webhook
🤖 Webhook para respuestas automáticas dinámicas de WhatsApp
GET /test
🔧 Verificar configuración y credenciales de Twilio
GET /crear-contenido
⚡ Crear contenido con botones interactivos (experimental)

⏰ Configuración Cron Jobs:

Mensaje Matutino: 0 7 * * * (7:00 AM) → /enviar-notificacion
Recordatorio: 45 17 * * * (5:45 PM) → /recordatorio

📱 Funcionalidades:

  • ✅ Notificaciones automáticas diarias
  • ✅ Respuestas dinámicas que cambian aleatoriamente
  • ✅ Soporte para múltiples estudiantes
  • ✅ Horario completo UTN (Lunes a Jueves 18:00-22:00)
  • ✅ Webhook configurado para interacciones
